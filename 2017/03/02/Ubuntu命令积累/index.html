Ubuntu安装中文字体

参考这里.
由于我是双系统，直接从win10里面C:/Windows/Fonts/目录下面，将所有字体文件都复制下来到

/home/shen/Downloads/font

再用cp命令：

sudo cp -r /home/shen/Downloads/font /usr/share/fonts/

最后：

sudo mkfontscale
sudo mkfontdir
sudo fc-cache -fv
