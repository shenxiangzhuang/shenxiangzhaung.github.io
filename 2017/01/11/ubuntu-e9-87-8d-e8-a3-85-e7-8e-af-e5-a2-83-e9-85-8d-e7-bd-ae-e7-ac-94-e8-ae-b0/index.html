&nbsp;&nbsp;&nbsp; 6.CMake安装

&nbsp;&nbsp;&nbsp;[ 官网](https://gitlab.kitware.com/cmake/cmake)下载，第一次编译

./bootstrap && make && make install
     最后因为不能创建一个文件，失败。估计是权限问题， 赋予权限并进行第二次尝试。 
sudo ./bootstrap && make && make install
    第二次尝试失败，还是权限问题。     所以将每次报错的文件夹赋予777权限，几步报错后，安装成功。     7.OpenCV安装配置     (1)按照[官网教程](http://docs.opencv.org/3.0-last-rst/doc/tutorials/introduction/linux_install/linux_install.html)[自备梯子...]安装，遇到无法安装libgkt2.0-dev，各种尝试发现，apt源除了问题。。。换回官方源，安装成功。。。     (2)报错-- ICV: Downloading ippicv_linux_20151201.tgz...，参考[这里](-- ICV: Downloading ippicv_linux_20151201.tgz...)     （3）[CMakeList.txt配置及测试代码](http://docs.opencv.org/3.2.0/db/df5/tutorial_linux_gcc_cmake.html)     8\.
